  • atom-live-server
    • html files require <body></body> so the server can inject the script
    • shortcut: command palette + StartS
    • shortcut (full): ctrl-shift-p + Atom Live Server: StartServer
    • shortcut: command palette + StopS
    • shortcut (full): ctrl-shift-p + Atom Live Server: StopServer
    • config file: .atom-live-server.json in root folder of atom
    • config options (command line): https://www.npmjs.com/package/live-server
  • atom-typescript
    • all-in-one atom TypeScript plugin
    • conflicts with vim-mode-plus ctrl-r, fix using keymap.cson
    • config file: tsconfig.json in any folder
    • suggested settings: 
      {
  "compilerOptions": {
    "strictNullChecks": true,
    "target": "es6",
    "sourceMap": true
  },
  "compileOnSave": true
}
    • config options: https://www.typescriptlang.org/docs/handbook/tsconfig-json.html
    • config compiler options: https://www.typescriptlang.org/docs/handbook/compiler-options.html
    • source map help: https://www.gamefromscratch.com/post/2014/05/27/TypeScript-debugging-in-Visual-Studio-with-IE-Chrome-and-Firefox-using-Source-Maps.aspx
